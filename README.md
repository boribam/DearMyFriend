# 디어프렌드(DEARFRIEND)
* 반려동물에 대한 잊기 쉬운 것들을 메모하고, 다른 사람들과 반려동물의 사진을 공유해보세요!

### ⭐ 주요 기능
* 반려동물의 종합백신, 건강검진 등의 일정과 반려동물의 건강상태를 메모하여 체크
* 내 주변에 있는 동물병원의 위치 찾기
* 게시물을 작성해 자신의 반려동물을 소개하고 다른 사용자와 공유
* 지역, 품종, 기간 별로 유기동물을 검색
